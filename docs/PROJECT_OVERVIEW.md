# 医学PDF结构化信息提取系统

## 项目简介
基于LLM的医学文献自动化结构化提取工具，支持指南、综述、其他类型医学文献。

## 核心功能
- 智能文档分类 (GUIDELINE/REVIEW/OTHER)
- 自适应提取策略 (短/中/长文档)
- 动态Few-shot学习
- 结构化JSON输出

## 技术栈
- 推理引擎: vLLM
- 基础模型: Qwen3-8B
- PDF解析: PyMuPDF

## 性能指标
| 文档类型 | 评分 | 耗时 |
|---------|-----|------|
| 短文档(≤15页) | 8.3/10 | 35s |
| 中文档(16-50页) | 6.5/10 | 45s |
| 长文档(>50页) | 6.0/10 | 90s |
